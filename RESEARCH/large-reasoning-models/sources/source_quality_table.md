# Source Quality Ratings

## Rating Scale

| Rating | Description | Criteria |
|--------|-------------|----------|
| **A** | Peer-reviewed / rigorous | Peer-reviewed papers, systematic reviews, NeurIPS/ICML/TMLR publications, detailed technical reports with methodology |
| **B** | Official vendor / reputable analyst | First-party vendor documentation, well-regarded engineering blogs with technical depth |
| **C** | Expert opinion / informed analysis | Industry blogs, curated wikis, tech journalism with sourced claims |
| **D** | Informal / preliminary | Personal blogs, Medium posts without rigorous sourcing, preprints without peer review |
| **E** | Anecdotal / speculative | Unsourced claims, forum posts, opinion without evidence |
