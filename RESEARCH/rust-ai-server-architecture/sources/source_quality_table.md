# Source Quality Ratings

## Rating Scale

| Rating | Meaning | Examples |
|--------|---------|---------|
| **A** | Peer-reviewed, official statements, government guidance | ACM papers, Rust Foundation, CISA |
| **B** | Reputable tech sources, primary repos, research reports | GitHub repos, HuggingFace docs, Zylos Research |
| **C** | Expert blogs, technical tutorials, vendor docs | dasroot.net, Calmops, Medium tech posts |
| **D** | Preliminary, opinion, single-practitioner blogs | DEV Community posts, Twitter threads |
| **E** | Anecdotal, theoretical, speculative | None used |
