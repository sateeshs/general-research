## Source Quality Rating System

| Rating | Description |
|--------|-------------|
| A | Peer-reviewed research, reproducible benchmarks, primary model documentation |
| B | Production frameworks with significant adoption (>10k stars or >1M downloads), technical blogs from authoritative sources |
| C | Expert analysis, tech press from reputable outlets, documented product behavior |
| D | Estimates, preliminary research, inferred from indirect evidence |
| E | Anecdotal, theoretical, speculative |
